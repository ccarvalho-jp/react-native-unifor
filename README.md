
# React Native - Unifor

Atividades práticas da disciplina de Desenvolvimento para Plataformas Móveis.

##  Atividades

### Atividade 1 - Apresentação Básica
Criação de tela com informações pessoais centralizadas.

### Atividade 2 - Espaçamento
Prática com `justifyContent` e espaçamento entre elementos.

### Atividade 3 - Formatação de Texto
Uso de texto com partes em negrito usando componentes aninhados.

### Atividade 4 - Cores
Aplicação de cores no background e nos textos.

### Atividade 5 - Informações Completas
Tela final com todas as informações incluindo GitHub.

##  Tecnologias

- React Native
- Expo Snack
- JavaScript
